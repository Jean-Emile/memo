Our Methodology

Access the sources → Develop on top → Contribute to it

We will be open-sourcing each component over time to ensure the quality of the source and its documentation so that it is accessible to the community. Subscribe to our mailing-list to get news.

Access

Code builds like a charm, tests pass on the most common architectures. A README is provided describing basic usage with code snippets.

Develop

All the tools (documentation, examples, comments, ...) are available for people to develop their own projects on top of our software.

Contribute

Contributions are welcomed through pull requests. We handle comments and bug reports.
