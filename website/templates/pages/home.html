Mount Volume

Create a volume and access files through a POSIX-compliant file system interface.­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­­

$> infinit-storage --create --s3 --name s3 --capacity 10TB
Created storage "s3".

$> infinit-network --create --storage s3 --storage local --name hybrid-cloud
Created network "hybrid-cloud".

$> infinit-volume --create --network hybrid-cloud --name company
Starting network "hybrid-cloud".
Creating volume root blocks.
Created volume "company".

$> infinit-volume --mount --name company --ountpoint /mnt/company/
Mounting volume "company".

$> cp -R Engineering /mnt/company/

Infinit is the result of 8 years of research and development

Overlay Network

Connects thousands of computing devices in peer-to-peer in order to scale and prevent any bottleneck or single point of failure.

Distributed Hash Table

Replicates encrypted blocks of data in order to ensure their availability and reliability in a decentralized manner.

File System

Provides a low-level virtual disk interface on top of the DHT, allowing for a hierarchical organization of files.
