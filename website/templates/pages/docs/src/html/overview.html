Overview

The Infinit technology allows for the creating for heterogeneous storage infrastructures by aggregating various storage resources and unifying them into a UNIX-like Infinit file system.


Infinit file systems benefit from several advantages over the centralized and distributed file systems commonly used.

Secure

All the files stored are cut into chunk and every chunk is encrypted (RSA/AES-256) with a unique key.

Control

Control the localisation of your data by selecting the storage resources/providers involved in the infrastructure.

Flexible

Manage your elastic storage capacity needs as you go by adding/removing storages without interrupting the service.

Natural

Access your files on any device, from anywhere at any time through a native file system interface.

Affordable

No need to acquire specific hardware/appliance or setup/maintain a complicated server-side installation.

Reliable

Every block of every file is dynamically replicated several times in order to ensure durability and availability of the data.

Scalable

With a peer-to-peer foundation, Infinit does not suffer single points of failure or bottlenecks.

High Performance

Benefit from a high-throughput UDP-based network protocol for faster data retrieval.

Fault Tolerant

Detects failures and automatically adapts the network to guarantee the reliability of the data.

On Demand

Access/stream the files on-demand without the need to store the files locally or download them before.

Access Control

Decide who can read/edit your files with fine-grain access control capabilities.

Interoperability

Runs on your dearest operating system (hopefully!): MacOS X, Windows, Linux, iOS and Android

Versioning

Browse through the different versions of a file.

Migration

Easily migrate your data from one provider to another whenever you need without any service interruption.

Multi Tenant

Designed from the ground up to support multiple tenants with the isolated and secure data.
