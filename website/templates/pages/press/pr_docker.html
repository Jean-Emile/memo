Press Release October 25, 2016

Infinit: the solution for persistent storage in containers

Quick Pitch

Techstars alumnus, Infinit (https://infinit.sh) is releasing today a Docker Volume Plugin making it easy to manage and scale persistent storage for Docker.


New York, NY: 25 October 2016

Container technologies have revolutionized the computing world in several ways, in particular by allowing developers and operators to break applications into several independent micro-services.

Docker became extremely powerful by its ability to spin off containers and scale them very quickly. Particularly powerful at first for making stateless applications elastic, Docker's popularity has now reached a point where developers want their applications to do more. And this requires state, hence storage one way or another: for databases, for media files served by websites, for logs and more.

As such, containers are in desperate need of a natively elastic and distributed storage system that adapts to the behavior of a Docker application. A recent survey among IT specialists found out that these storage issues are holding back the fast and growing adoption of container technology in the enterprise.

The team behind the startup Infinit (https://infinit.sh) has been working hard this past year to provide a storage platform, with scalability and resilience in mind.

One specificity of the Infinit Storage Platform is its decentralised architecture i.e peer-to-peer (unlike the master slave model) that does away with single points of failure and bottlenecks while allowing for a better scalability.

Infinit is also highly programmatic allowing developers to customize the behavior of the storage infrastructure through a set of properties: encryption, deduplication, compression, redundancy, versioning, etc.. The storage platform is accessible through a set of interfaces from block device, object store and POSIX-compliant filesystems.

This architecture allows Infinit to match exactly the behaviour of containerized applications. For instance, a web application could be split into multiple micro-services. Each of these services could rely on a specifically customised Infinit storage backend: a block interface without encryption and compression for a database container like mongodb, a S3 compatible object store for storing images of a Wordpress frontend and more.

Infinit provides two ways to scale the storage component: (i) by scaling along with the application cluster e.g Docker Swarm, offering simplicity over control, essentially providing an hyper-converged system (scaling both storage, network and compute together), or (ii) by letting the administrator control the scalability of the components independently, leading to an hyper-scaled architecture.

No matter your configuration, thanks to the new Infinit Docker Volume Plugin, the storage platform will always be accessible to Docker containers.

This new and more modern wave of software-defined storage products like Infinit is making its way to the entreprise market, in particular because well adapted to more dynamic container environments. While providing a free community edition for developers, Infinit also provides entreprise licences with advanced functionalities.

Enterprises have learned that expensive appliances, non-scalable protocols, monolithic setups and non-programmable components are just too far off to keep pace with ever-changing business requirements. Infinit can now help them build or transform their storage infrastructure in a more durable, secure and evolving way.
